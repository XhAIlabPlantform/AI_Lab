使用场景
公安视频监控
针对中远距离的人脸监控识别，整合人脸识别、人体识别、实时监控等功能，为刑侦人员提供快速准确的取证手段
人流量统计
利用人体检测技术，你的应用能够轻松识别出图像或视频中人体并计数，从而在零售商业、机场或学校等场所进行基于人体的人流量或到场率统计
远程实名认证
识别客户上传的身份证信息，并进行证件图片真实性鉴别，用于远程身份核实、信息绑定场景。已经被互联网金融、电商等行业广泛采用
人脸解锁
运用人脸比对技术，您的应用能够比对实拍照片与底图，核验用户身份。这被广泛应用于打造刷脸登录或远程核身功能中。
相机美化
基于五官及轮廓关键点识别，对人脸特定位置进行修饰加工，实现人脸的特效美颜、特效相机、贴片等互动娱乐功能
驾驶员监控系统
基于人脸检测、疲惫检测、注意力检测，兼顾了安全、驾驶辅助
舆情监控
通过对需要舆情监控的实时文字数据流进行情感倾向性分析，把握用户对热点信息的情感倾向性变化
智能客服
集成知识问答接口，可准确回答各知识领域的专业问题。拒绝生硬回答，提升用户体验
